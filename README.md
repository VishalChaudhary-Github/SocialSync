# SocialSync

---

### Description - SocialSync is basically a social media clone project built completely using django.

#### For the Frontend - Django template engine is used and HTML, Bootstrap and CSS grid.
#### For the backend - API endpoints are defined using Django Urls, Http requests are processed using Django Views, Database is queried using Django ORM.

#### Commands to use - First Clone the repository then;
After activating the env in the project directory.

1. `python manage.py runserver` - To start a local server on 8000 port.
2. Feel free to drop any pull requests. They will be reviewed and merge accordingly


### Basic Usage - 
1. User can create account.
2. Log In using that account.
3. Setup their profile like uploading profile picture, changing password, bio etc.
4. Adding Post.
5. Follow, Unfollow other users.
6. Like followed users post.
